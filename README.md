# HVG - 数字创意科技公司网站

一个现代化的科技公司网站，展示系统开发、社交媒体运营和全案策划服务。

✨ **现代设计**
- 深色主题设计，使用 Tailwind CSS
- 渐变色彩和玻璃态效果
- 响应式布局，完美适配各种设备

🎯 **交互功能**
- 平滑滚动导航
- 移动菜单支持
- 动态计数器动画
- 表单验证和提交
- 元素进入视口时的动画效果

📱 **完全响应式**
- 移动设备优化
- 平板设备兼容
- 桌面完整体验

🚀 **性能优化**
- 轻量级代码
- 快速加载速度
- 无外部依赖（仅使用 Tailwind CDN）

## 使用方法

### 本地开发

1. **克隆或下载项目**
   ```bash
   cd hvg-website
   ```

2. **使用 Live Server 打开**
   - 在 VS Code 中安装 "Live Server" 扩展
   - 右键点击 `index.html` 选择 "Open with Live Server"

3. **或直接打开**
   - 直接在浏览器中打开 `index.html` 文件

### 自定义内容

#### 修改公司信息
编辑 `index.html` 中的以下部分：
- 公司名称：搜索 "HVG"
- 联系邮箱：搜索 "info@hvg.com"
- 电话号码：搜索 "+86 (010)"
- 地址：搜索 "北京市朝阳区"

#### 修改服务内容
在"服务区域"部分修改三个服务卡片的内容。

#### 添加新的作品
在"作品展示"部分复制现有的项目卡片并修改内容。

#### 更改颜色主题
修改 `styles.css` 中的 CSS 变量：
```css
:root {
    --cyan: #06b6d4;
    --blue: #3b82f6;
    /* 其他颜色... */
}
```

## 部署

### 部署到 GitHub Pages

1. 创建一个 GitHub 仓库
2. 上传项目文件
3. 在仓库设置中启用 GitHub Pages
4. 选择 main 分支作为源

### 部署到其他平台

- **Netlify**: 直接拖放文件夹
- **Vercel**: 连接 GitHub 仓库
- **传统服务器**: 通过 FTP 上传文件

## 常见问题

**Q: 如何修改颜色？**
A: 修改 `styles.css` 中的 CSS 变量或直接编辑 `index.html` 中的 Tailwind 类名。

**Q: 如何添加页面？**
A: 创建新的 HTML 文件并复制导航栏和页脚代码，然后添加新的内容部分。

**Q: 如何集成数据库？**
A: 使用 Node.js + Express 后端，或使用 Firebase 等 BaaS 平台。

**Q: 如何提高网站速度？**
A: 优化图片、使用 CDN、启用缓存、最小化代码等。
